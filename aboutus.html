ABOUT THIS SITE

This task was created for class and was aimed at assessing students' proficiency in using various tools and frameworks to achieve a shared objective over a two-week period. Students were tasked with integrating their knowledge of HTML, CSS, Bootstrap, and Git to create a comprehensive project that demonstrated their understanding and application of these concepts.

HTML
Students utilized HTML to structure the content of their project. This included defining the layout of web pages, organizing text, images, and other media into a cohesive structure that forms the foundation of web development.

CSS
CSS was employed to style the HTML elements, ensuring visual consistency and enhancing the aesthetic appeal of the project. Students applied CSS rules to control typography, colors, spacing, and layout, thereby customizing the appearance of their web pages according to design specifications.

Bootstrap
Bootstrap, a popular front-end framework, was integrated into the project to expedite development and ensure responsiveness. Students leveraged Bootstrap's pre-built components, grid system, and CSS classes to create a responsive and mobile-friendly layout without reinventing the wheel.

Git
Git, a distributed version control system, played a crucial role throughout the project's lifecycle. Students utilized Git to manage their codebase efficiently, track changes, collaborate with teammates through branching and merging, and maintain a history of project iterations. This fostered good coding practices and teamwork skills essential for real-world software development scenarios.
